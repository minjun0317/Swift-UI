## 생산성 (알면 개편함)
- Cmd + 클릭 : 깊은 선택 // 그룹 안 바로 선택
- Enter : 내부 레이어 진입
- Shift + Enter : 상위로 나가기
- Cmd + / : 기능 검색

---

## 이동 / 복사
- 방향키 : 1px 이동
- Shift + 방향키 : 10px 이동
- Option + 방향키 : 간격 조절
- Cmd + D : 마지막 행동 반복 // 정렬/복사 반복 가능

---

## 정렬
- Cmd + Option + T : 위 정렬
- Cmd + Option + B : 아래 정렬
- Cmd + Option + L : 왼쪽 정렬
- Cmd + Option + R : 오른쪽 정렬
- Cmd + Option + H : 가로 중앙
- Cmd + Option + V : 세로 중앙

---

## 크기 / 비율
- Shift + 드래그 : 비율 유지
- Option + 드래그 : 중심 기준 확대
- K : 스케일 툴 // 전체 비율 유지 변경

---

## 텍스트
- Cmd + B : 볼드
- Cmd + I : 이탤릭
- Cmd + Shift + < / > : 크기 조절
- Option + 방향키 : 자간 조절

---

## 색상 / 스타일
- I : 스포이드 // 색상 추출
- Cmd + Option + C : 스타일 복사
- Cmd + Option + V : 스타일 붙여넣기

---

## 컴포넌트 (중요)
- Cmd + Option + K : 컴포넌트 생성
- Option + 드래그 : 인스턴스 복제
- Cmd + 클릭 : 인스턴스 내부 수정

---

## 오토 레이아웃 (핵심)
- Shift + A : 적용
- 방향 전환 : 오른쪽 패널에서 설정
- Padding / Gap : 간격 자동 관리

---

## 뷰 / 정렬 보조
- Cmd + ; : 가이드 보기/숨김
- Cmd + ' : 픽셀 그리드
- Option : 거리 측정

---

## 꿀팁
- Cmd + D 반복 활용하면 복사 작업 속도 2배
- 오토레이아웃 안 쓰면 UI 작업 시간 낭비
- 컴포넌트 안 쓰면 수정 지옥 시작
