# Figma 단축키 정리

## 기본 도구
- V : 선택
- F : 프레임 생성
- R : 사각형
- O : 원
- T : 텍스트
- L : 선
- P : 펜

---

## 레이어
- Cmd + G : 그룹
- Cmd + Shift + G : 그룹 해제
- Cmd + D : 복제
- Option + 드래그 : 빠른 복사

---

## 레이어 순서
- Cmd + ] : 앞으로 // 한 단계
- Cmd + [ : 뒤로 // 한 단계
- Cmd + Shift + ] : 맨 앞으로
- Cmd + Shift + [ : 맨 뒤로

---

## 정렬 / 레이아웃
- Shift + A : 오토 레이아웃 // 자동 정렬
- Option : 간격 확인 // 요소 사이 거리 표시
- Shift + 드래그 : 이동 고정 // 수평/수직

---

## 보기
- Cmd + + : 확대
- Cmd + - : 축소
- Cmd + 0 : 화면 맞춤
- Cmd + 1 : 100%

---

## 기타
- Cmd + / : 검색
- Cmd + Z : 실행 취소
- Cmd + Shift + Z : 다시 실행

---

## 참고
- 위 레이어가 앞에 보임
- 프레임 안에서 작업 권장
